Sway在无显示管理器的情况下实现开机自动登录

分享一下SwayWM在无显示管理器(Display Manager)的情况下如何实现自动登录。我一开始是设置了fish登入tty1后自动进入sway，参考SwayWiki

PS: 我使用的是Arch Linux，如果是其他发行版操作可能会有区别。

具体操作步骤参考了ArchWiki进行的。
首先创建目录，然后切换到该目录。

sudo mkdir -p /etc/systemd/system/getty@tty1.service.d
cd /etc/systemd/system/getty@tty1.service.d

接着新建一个文件,并编辑

sudo touch skip-prompt.conf
sudo vim skip-prompt.conf
[Service]
Environment=XDG_SESSION_TYPE=wayland
ExecStart=
ExecStart=-/usr/bin/agetty --skip-login --nonewline --noissue --autologin username --noclear %I $TERM

这里注意把username改为你实际要登录的用户名，并且注意如果你使用的是x11的窗口管理器（例如i3、dwm）等，则要将XDG_SESSION_TYPE=wayland替换为XDG_SESSION_TYPE=x11。
